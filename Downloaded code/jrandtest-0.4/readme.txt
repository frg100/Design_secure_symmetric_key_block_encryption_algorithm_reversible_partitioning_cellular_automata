Install
~~~~~~~
	0. (install j2re / java runtime or java sdk)
	1. unzip package
	2. Gui interface - execute run.bat or run.sh
	3. CmdLine interface - execute runCmdLine.bat or runCmdLine.sh

Bugs
~~~~

If you think you found a bug, please use the mailing lists or the bug
tracking system on jrandtest.sourceforge.net to report it. Do not forget
to include all information about your system.
